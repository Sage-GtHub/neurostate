# British English Standards for NeuroState

This guide ensures consistent British English across the entire website and application.

## Spelling Conversions Applied

### -ize to -ise
- optimize → optimise
- personalize → personalise
- recognize → recognise
- analyze → analyse
- customize → customise
- organize → organise

### -or to -our  
- color → colour
- behavior → behaviour
- favor → favour

### -er to -re
- center → centre

### Other Common Terms
- organization → organisation (when referring to companies/entities, keep "Organization" for schema.org structured data)
- gray → grey

## Implementation Notes

- All user-facing text must use British spellings
- Technical terms in code (variables, functions) can remain American for consistency with libraries
- Schema.org structured data uses "@type": "Organization" as per standard
- All mobile optimization includes touch-friendly elements (min-h-[44px] for tap targets)
